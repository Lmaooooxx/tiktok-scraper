# TikTok Today – Lấy Video Đăng Hôm Nay

## Mô tả
Project này giúp lấy danh sách video được đăng **hôm nay** của một hoặc nhiều user TikTok.  
Hỗ trợ cả:
- **CLI**: giao diện dòng lệnh đơn giản  
- **Web UI**: giao diện trực quan bằng **Streamlit**

Thời gian lấy video theo **UTC+7 (giờ Việt Nam)**.
